# Codsoft_Face-Detection
A real-time face detection system built using Python, OpenCV, and Tkinter that detects faces via webcam and displays simulated mood, age group, gender, and analytics in a modern HUD-style interface.

🚀 Features
🎯 Real-time face detection using Haar Cascades
😊 Simulated mood detection with emojis & intensity
👤 Age group & gender prediction (randomized demo)
🧠 Face tracking with unique IDs
📊 Live HUD (FPS, detections, uptime, snapshots)
📸 Snapshot capture feature
🖥️ Dark-themed GUI using Tkinter
👁️ Eye & smile landmark detection
🎨 Stylish overlays (bounding brackets, scanlines)
📜 Mood activity log

🛠️ Technologies Used
Python
OpenCV
Tkinter
NumPy
Pillow (PIL)
